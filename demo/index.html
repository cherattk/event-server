Warning : before starting the demo, make sure that the event server is running.
Usage :
  1. Go to [Event-Server GUI] >> [Setting] and register a Service and the Event that it triggers.
  2. Attach the following 3 Listeners Endpoints to the Event :
    http://localhost:3001,
    http://localhost:3002 ,
    http://localhost:3003 .
  3. Copy the generated event ID and past it in the the Event ID input.
  4. Write a Message and Submit.

  5. The form will make an HTTP POST Request to the Event-Server located by default at :
